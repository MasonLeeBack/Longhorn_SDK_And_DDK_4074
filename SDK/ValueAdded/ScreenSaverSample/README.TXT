Do not try to rebuild this screen saver and install without adding category information. 
To learn more about how to write a screen saver and how to add category information, 
use the SDK and search under the term 'Screen saver'.

---------------------------------------------------------------------------
		QUICK STEPS TO INSTALL THIS SCREENSAVER
---------------------------------------------------------------------------
To install this screensaver from a longhorn SDK command console, type:

ssutils BouncingBallsScreenSaver.exe BouncingBallsScreenSaver.manifest

To uninstall this screensaver from a longhorn SDK command console, type:

ssutils -uninstallmanifest BouncingBallsScreenSaver.manifest

-----------------------------------------------------------------------------

To install this screensaver from a regular command console:

copy ssutils.exe and csutils.exe (most likley in C:\Program Files\Microsoft Longhorn SDK\Bin) into this directory.  Type:

ssutils BouncingBallsScreenSaver.exe BouncingBallsScreenSaver.manifest

uninstall with the command:

ssutils -uninstallmanifest BouncingBallsScreenSaver.manifest
