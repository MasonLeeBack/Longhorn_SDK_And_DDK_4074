Sample WIA Microdriver

[This is preliminary documentation and subject to change.]

SUMMARY

This sample shows how to write a Windows Image Acquisition (WIA) Microdriver for a flatbed scanner. The Microdriver architecture is functionally reduced from the WIA Minidriver architecture, but supports the primary scanning functions. Documentation on WIA Microdrivers is provided in the Microsoft® Windows® Driver Development Kit (DDK). This sample driver can be used as a starting point for your development, but your driver should access the scanner hardware through one of the Microsoft Windows kernel drivers.

This sample will not work with the Microsoft Windows 98 or Microsoft Windows 2000 operating systems.

BUILDING THE SAMPLE

To build the sample:

1.     On the Start menu, point to Windows DDK, and then click Free Build Environment or Checked Build Environment to open a command window that sets basic environment variables that are needed to build drivers.

2.     Change to the src\wdm\wia\microdrv directory that contains the sample driver source code.

3.     Execute the build command. After the build is completed, a driver named Testmcro.dll will be located in a Microdrv subdirectory.

USING THE SAMPLE

Before using Testmcro.dll, install it:

  1. Copy Testmcro.dll from the subdirectory under Microdrv to a new, empty directory.
  2. Copy Testmcro.inf from the Microdrv directory to the directory that holds Testmcro.dll.
  1. On the Start menu, click Run, and then type rundll32 sti_ci AddDevice in the Open box.
  2. When prompted for the scanner or camera that you want to install, click Have Disk, and then navigate to the folder created in step 1 to install the driver.
  3. Click OK, Next, or Finish until the wizard closes.

Use one of these options to test Testmcro.dll:

·       Use the Scanners and Cameras Wizard.

·       Use the namespace extension accessible by opening the camera with Windows Explorer.

·       Use the Wiainfo2 application. For more information, see the Windows DDK.

·       Use Microsoft Paint, a native WIA application.

·       Use any TWAIN-compliant application.
