SAP Scenario Generator

[This is preliminary documentation and subject to change.]

Prerequisite

Install Windows Media Player version 7 or greater.

SUMMARY

This application generates five SAP scenarios for testing purposes. The five scenarios are based on the three SAP parameters : 1) SAP certs 2) Digital output 3) Minimum security level .

Input is a clear content file. Can be a wma or asf format.

INSTALLING THE SAMPLE

Launch the SAPGen.exe. The input file can be a .wma or .asf file. Select the input file and click on the Generate button to generate the five output files corresponding to the five scenarios along with the licenses needed to play the content. A html file is launched pointing to these five files along with the description of their expected behavior. This result file SAPScenarioResult.html and the five wma files are created in the same directory as the input file.

In case where the tool fails with handle = 80048009 please get the updated version from http://go.microsoft.com/fwlink/?LinkId=3458

PROGRAMMING TOUR

The application uses different KID and Key pair to package the content and generate license in each scenario.

Scenario 1 : Certs not required. Digital out and security level do not play a part. This should play even if the driver is not certified. Filename is CertfalseDigfalseSec1100.wma.

Scenario 2 : Certs required, disabled digital out, Mindrmlevel 1100. This should play only if the driver is certified and digital out MUST be disabled by the driver. Filename is CerttrueDigfalseSec1100.wma.

Scenario 3 : Certs required, enabled digital out, Mindrmlevel 1100. This should play only if the driver is certified and digital out can be enabled by the driver. Filename is CerttrueDigtrueSec1100.wma.

Scenario 4 : Certs required, disabled digital out, Mindrmlevel 15000. This should NOT play and player MUST return an error 'Certified drivers are....'. Filename is CerttrueDigfalseSec15000.wma.

Scenario 5 : Certs required, enabled digital out, Mindrmlevel 15000. This should NOT play and player MUST return an error 'Certified drivers are....'. Filename is CerttrueDigtrueSec15000.wma.
