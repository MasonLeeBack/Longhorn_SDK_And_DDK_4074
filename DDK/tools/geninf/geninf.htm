INF File Creation Wizard

 

SUMMARY

GenINF is a GUI-based tool that guides the user through the process of creating Microsoft® Windows® 2000 and Microsoft® Windows® Whistler device driver .inf files. The tool is still under development, so please note the Current Status and Limitations sections listed below.

Current Status

GenINF is still under development. The version included in this release of the Windows Whistler DDK should be considered preliminary.

The tool does not include support for all device classes, but specific support for some device classes is under development. Currently, GenINF natively supports the following device classes:

·         1394 host bus controllers (class = 1394)

·         Battery devices (class = Battery)

·         CD-ROMs (class = CDROM)

·         Disk drives (class = DiskDrive)

·         Dot4 devices (class =Dot4)

·         Floppy disk controllers (class = FDC)

·         Floppy disk drives (class = FloppyDisk)

·         Hard disk controllers (class = HDC)

·         Human input devices (class = HID)

·         Memory technology drivers (class = MTD)

·         Multifunction devices (class = Multifunction)

·         Multi-port serial adapters (class = MultiportSerial)

·         PCMCIA adapters (class = PCMCIA)

·         Ports (class = Ports)

·         Smart card readers (class = SmartCardReader)

·         Storage volumes (class = Volume)

·         System devices (class = System)

·         USB (class = USB)

In addition to the device classes mentioned above, GenINF also provides some device-specific support to the following classes:

·         Display adapters (class = Display)

·         Keyboards (class = Keyboard)

·         Multimedia devices (class = Media)

·         Mice (class = Mouse)

·         SCSI and RAID controllers (class = SCSIAdapter)

GenINF does not always generate fully functional .inf files. However, it should generate valid files for most devices of the listed device classes. GenINF also provides context-sensitive help in order to aid users with questions on either the wizard or the INF format.  Devices of class SCSIAdapter may also use the tool to create a txtsetup.oem file.

The tool has not been extensively tested. Use it at your own risk.

Running the Tool

GenINF is located in the <DDKRoot>\Tools subdirectory. Because it is GUI-based, it can be executed from within the Windows 2000 User Interface, or from the Command Prompt. GenINF takes no arguments. Help can be accessed at any time within the tool by pressing the Help button at the bottom of the wizard.

Limitations

GenINF supports the creation of .inf files for Windows 2000/Windows Whistler only. It does not create .inf files for Microsoft Windows 95, Microsoft Windows 98, Microsoft Windows Millennium or Microsoft Windows NT® 4.0.

GenINF can be used to create multi-architecture .inf files (x86/IA64).

Users should review all .inf files created by GenINF, and modify them as required. Users should expect to add additional .inf directives that are device-class-specific or that contain device-class-specific information.
