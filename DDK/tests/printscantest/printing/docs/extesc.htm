ExtEsc Test

Introduction

The ExtEsc.exe test does robustness and some basic functionality testing of drivers that handle the ExtEsc API.

Scope – Longhorn Test

ExtEsc is a Windows Longhorn test that is part of the Microsoft Windows codename “Longhorn” Driver Kit (LDK).

  

Global Information

Background Information

Primary Operating System

Windows Longhorn

Supported Operating Systems

Microsoft Windows 2000, Microsoft Windows XP, Microsoft Windows Server™ 2003.

Device\Driver Category

GDI Drivers—display, print, and fax drivers.

Related DDK Topics

DDK\Display and Print Devices\Design Guide\Graphic Drivers

Friendly Name

Printer ExtEscape Robustness Test

Display ExtEscape Robustness Test

Internal Name

ExtEsc.exe

Description

This test sends random data to the ExtEscape including bad sizes.  It also performs simple functionality testing for supported escapes. The test determines if bad or good data will cause faults in system components or services.  For example, a BSOD computer or crash spooler service.

Theory of Operation

This is a console based test that slams the ExtEscape API with random data, bad sizes, and simple, good cases. 

Installation

Installation Steps

Copy the binary to the local machine. The test requires WTTLOG.dll

File Location

The location of the files to be installed for this test:

<LDK dir>\tests\printscantest\printing\<Arch>\ExtEsc.exe

These files will be installed to:

<LDK dir>\tests\printscantest\printing\<Arch>\ExtEsc.exe

Removal Steps

The files are deleted by the Windows LDK uninstall program.  Users may need to delete the log files. 

Environment Requirements

Required Software

Windows XP, Windows Server 2003, Windows Longhorn or later.  Printer driver or display driver. 

Diagnosability

Setup Verification

This tool doesn’t provide verification that setup was completed correctly because it’s just a single .exe file that needs to be copied.  

Test Log Diagnosis

The only failure should be if the computer BSOD or spooler service crashes.  The crash itself will need to be analyzed.
